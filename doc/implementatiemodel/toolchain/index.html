Afhankelijkheden

Dit document beschrijft een implementatiemodel, in dit geval Data Specifications. Een implementatiemodel is een toepassing van vocabularia voor een concreet doel. Het beschrijft in alle detail (zoals kardinaliteiten, codelijsten, datatypes, ...) de gewenste datastructuur.
